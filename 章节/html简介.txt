1. 什么是html  

	HTML 是用来描述网页的一种语言。

		HTML 指的是超文本标记语言 (Hyper Text Markup Language) 
		HTML 不是一种编程语言，而是一种标记语言 (markup language) 
		标记语言是一套标记标签 (markup tag) 
		HTML 使用标记标签来描述网页 

2. .htm 和.html
	当保存 HTML 文件时，既可以使用 .htm 也可以使用 .html 文件后缀。
	这是历史遗留问题，因为过去的很多软件只允许三个字母的文件后缀。

	不过对于新的软件，使用 .html 完全没有问题。
	
3. html标签
	
	HTML 标记标签通常被称为 HTML 标签 (HTML tag)。
	HTML 文档和 HTML 元素是通过 HTML 标签进行标记的
	
		HTML 标签是由尖括号包围的关键词，比如 <html> 
		HTML 标签通常是成对出现的，比如 <b> 和 </b> 
		标签对中的第一个标签是开始标签，第二个标签是结束标签 
		开始和结束标签也被称为开放标签和闭合标签 
		开始标签是被尖括号包围的元素名 
		结束标签是被尖括号包围的斜杠和元素名 
		某些 HTML 元素没有结束标签，比如 <br /> 
		
注释：开始标签的英文翻译是 start tag 或 opening tag，结束标签的英文翻译是 end tag 或 closing tag。


4. HTML 文档 = 网页
	HTML 文档描述网页 
	HTML 文档包含 HTML 标签和纯文本 
	HTML 文档也被称为网页 
	Web 浏览器的作用是读取 HTML 文档，并以网页的形式显示出它们。浏览器不会显示 HTML 标签，而是使用标签来解释页面的内容：

5.html元素
	
	5.1 HTML 元素指的是从开始标签（start tag）到结束标签（end tag）的所有代码
		开始标签 							元素内容 				结束标签 
		<p> 								This is a paragraph 	</p> 
		<a href="default.htm" >				 This is a link 		</a> 
		<br /> 
	
	5.2 HTML 元素语法
	
		HTML 元素以开始标签起始 
		HTML 元素以结束标签终止 
		元素的内容是开始标签与结束标签之间的内容 
		某些 HTML 元素具有空内容（empty content） 
		空元素在开始标签中进行关闭（以开始标签的结束而结束） 
		大多数 HTML 元素可拥有属性 
		
	5.3 嵌套的 HTML 元素
	
		大多数 HTML 元素可以嵌套（可以包含其他 HTML 元素）。

		HTML 文档由嵌套的 HTML 元素构成。
		
	5.4 空的 HTML 元素
	

		没有内容的 HTML 内容被称为空元素。空元素是在开始标签中关闭的。

		在 XHTML、XML 以及未来版本的 HTML 中，所有元素必须被关闭。

		在开始标签中添加斜杠，比如 <br />，是关闭空元素的正确方法，HTML、XHTML 和 XML 都接受这种方式。

		即使 在开始标签中不添加斜杠 如<br> 在所有浏览器中都是有效的，但建议在开始标签中添加斜杠 <br /> 。

6. html属性
		
		6.1 HTML 标签可以拥有属性
			属性总是以名称/值对的形式出现
			属性总是在 HTML 元素的开始标签中规定
		
		6.2 HTML 提示：使用小写属性
			属性和属性值对大小写不敏感。
			不过，万维网联盟在其 HTML 4 推荐标准中推荐小写的属性/属性值。
			而 XHTML 要求使用小写的属性/属性值
			
		6.3 始终为属性值加引号
		属性值应该始终被包括在引号内。
		一般是双引号,只用单引号也可以,但是在某些个别的情况下，比如属性值本身就含有双引号，必须使用单引号

		
7.注意事项

	7.1 不要忘记结束标签
	
		如果忘记使用结束标签 大多数浏览器也会正确显示 .但有时候会产生不可预料的结果或问题,如<b>没有结束标签的话,自<b>以后所有文本都加粗
		注释：未来的 HTML 版本不允许省略结束标签
		
	7.2 HTML 提示：使用小写标签
	
		HTML 标签对大小写不敏感：<P> 等同于 <p>。

		因为万维网联盟（W3C）在 HTML 4 中推荐使用小写，而在未来 (X)HTML 版本中强制使用小写。
